Terrible Tarot

Terrible Tarot is a message generation programme that utilises JavaScript to display a user's fortune. It generates a new, random fortune each time it is run.

Drag-A-Doodle

Based on the classic Etch-A-Sketch drawing game, Drag-A-Doodle lets users set a custom grid size for the drawing board, draw and erase with the mouse, pick and change colors, and reset the board once done.

Rock, Paper Scissors

A browser version of the classic Rock, Paper, Scissors game. This version features user choice buttons, a visual game board, and winner pop-ups.
